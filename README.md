# Team Task Manager

A full-stack task management application built with React, Node.js, Express, and MongoDB. Users can create, manage, update, and delete tasks with priorities, statuses, and deadlines.

---

## 🌐 Live Demo

🔗 Live Website: https://team-task-manager-ufxp.onrender.com/

---

## 🚀 Features

- Create New Tasks
- Update Task Status
- Delete Tasks
- Task Priority Management
- Due Date Support
- Responsive Design
- REST API Integration
- MongoDB Database

---

## 🛠️ Tech Stack

### Frontend

- Axios

### Backend
- Node.js

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/your-username/team-task-manager.git
```

---

## 🔥 Backend Setup

```bash
cd server
npm install
npm run dev
```

Create a `.env` file inside the server folder:

```env
PORT=5001
MONGO_URI=your_mongodb_connection
```

---

## 💻 Frontend Setup

```bash
cd client
npm install
npm start
```

---

## 📡 API Endpoints

| Method | Endpoint | Description |
|---|---|---|
| GET | `/api/tasks/all` | Fetch all tasks |
| POST | `/api/tasks/create` | Create a task |
| PUT | `/api/tasks/update-status/:id` | Update task status |
| DELETE | `/api/tasks/delete/:id` | Delete task |
